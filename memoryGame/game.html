Memory game!

Gra memory polega na odnajdywaniu par takich samych kart. Gracz odsłania 2 karty. Jeśli są to takie same karty, zostaną one zdjęte z planszy jeśli nie, karty będą odwrócone z powrotem (po ok. 2 sekundach). Celem gracza jest zdjęcie wszystkich kart przy możliwie najmniejszej liczbie odsłon i jak najkrótszym czasie, co będzie miało wpływ na wynik końcowy. W grze liczona jest ilość błędnych odsłon.
